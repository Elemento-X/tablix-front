Na hora de exportar uma planilha, sempre aparece a dúvida: salvar como CSV ou XLSX? Os dois guardam dados em tabela, mas servem a propósitos diferentes. Escolher errado pode custar fórmulas perdidas ou acentos quebrados. Veja como decidir.

## O que é cada formato

**CSV (Comma-Separated Values)** é um arquivo de texto puro: cada linha é uma linha da tabela, e as colunas são separadas por um delimitador (vírgula ou ponto e vírgula). É simples, leve e lido por praticamente qualquer sistema.

**XLSX** é o formato do Excel moderno. Além dos dados, ele guarda fórmulas, formatação, múltiplas abas, gráficos e tipos de dados (datas, números, moeda).

## Quando usar CSV

- Para **importar e exportar** dados entre sistemas diferentes.
- Quando o destino é um banco de dados, uma API ou outra ferramenta.
- Para arquivos **leves** com muitas linhas e sem necessidade de formatação.

A limitação: CSV não guarda fórmulas, formatação nem abas. É só o dado cru.

## Quando usar XLSX

- Quando você precisa de **fórmulas, gráficos ou várias abas**.
- Para relatórios que serão **lidos por pessoas**, com formatação visual.
- Quando os tipos de dados (datas, moeda) precisam ser preservados.

A limitação: arquivos maiores e nem todo sistema importa XLSX com facilidade.

## Resumo rápido

| Critério | CSV | XLSX |
| --- | --- | --- |
| Fórmulas e formatação | Não | Sim |
| Várias abas | Não | Sim |
| Tamanho do arquivo | Menor | Maior |
| Compatibilidade entre sistemas | Altíssima | Média |
| Ideal para | Trocar dados | Relatórios |

## Perguntas frequentes

**Posso converter CSV em XLSX e vice-versa?**
Sim. O Excel e a maioria das ferramentas de planilha abrem um e salvam no outro. Lembre que ao salvar como CSV você perde fórmulas e formatação.

**Qual formato ocupa menos espaço?**
CSV, por ser texto puro. Para grandes volumes sem formatação, ele é mais leve.
