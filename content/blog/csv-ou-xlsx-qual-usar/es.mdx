A la hora de exportar una hoja de cálculo, siempre surge la duda: ¿guardar como CSV o XLSX? Ambos guardan datos en tabla, pero sirven a propósitos distintos. Elegir mal puede costarle fórmulas perdidas o acentos rotos. Vea cómo decidir.

## Qué es cada formato

**CSV (Comma-Separated Values)** es un archivo de texto plano: cada línea es una fila de la tabla, y las columnas se separan por un delimitador (coma o punto y coma). Es simple, ligero y lo lee prácticamente cualquier sistema.

**XLSX** es el formato del Excel moderno. Además de los datos, guarda fórmulas, formato, varias hojas, gráficos y tipos de datos (fechas, números, moneda).

## Cuándo usar CSV

- Para **importar y exportar** datos entre sistemas distintos.
- Cuando el destino es una base de datos, una API u otra herramienta.
- Para archivos **ligeros** con muchas filas y sin necesidad de formato.

La limitación: el CSV no guarda fórmulas, formato ni hojas. Es solo el dato crudo.

## Cuándo usar XLSX

- Cuando necesita **fórmulas, gráficos o varias hojas**.
- Para informes que serán **leídos por personas**, con formato visual.
- Cuando los tipos de datos (fechas, moneda) deben preservarse.

La limitación: archivos más grandes, y no todo sistema importa XLSX con facilidad.

## Resumen rápido

| Criterio | CSV | XLSX |
| --- | --- | --- |
| Fórmulas y formato | No | Sí |
| Varias hojas | No | Sí |
| Tamaño del archivo | Menor | Mayor |
| Compatibilidad entre sistemas | Altísima | Media |
| Ideal para | Intercambiar datos | Informes |

## Preguntas frecuentes

**¿Puedo convertir CSV en XLSX y viceversa?**
Sí. Excel y la mayoría de las herramientas de hojas de cálculo abren uno y guardan en el otro. Recuerde que al guardar como CSV pierde fórmulas y formato.

**¿Qué formato ocupa menos espacio?**
CSV, por ser texto plano. Para grandes volúmenes sin formato, es más ligero.
