Beim Exportieren einer Tabelle stellt sich immer dieselbe Frage: als CSV oder XLSX speichern? Beide speichern Daten in einer Tabelle, dienen aber unterschiedlichen Zwecken. Falsch zu wählen kann verlorene Formeln oder kaputte Umlaute kosten. So entscheiden Sie.

## Was jedes Format ist

**CSV (Comma-Separated Values)** ist eine reine Textdatei: Jede Zeile ist eine Tabellenzeile, und Spalten werden durch ein Trennzeichen (Komma oder Semikolon) getrennt. Es ist einfach, leicht und wird von praktisch jedem System gelesen.

**XLSX** ist das Format des modernen Excel. Neben den Daten speichert es Formeln, Formatierung, mehrere Registerblätter, Diagramme und Datentypen (Datum, Zahlen, Währung).

## Wann CSV verwenden

- Zum **Importieren und Exportieren** von Daten zwischen verschiedenen Systemen.
- Wenn das Ziel eine Datenbank, eine API oder ein anderes Tool ist.
- Für **leichte** Dateien mit vielen Zeilen ohne Formatierungsbedarf.

Die Einschränkung: CSV speichert keine Formeln, keine Formatierung und keine Registerblätter. Es sind nur die Rohdaten.

## Wann XLSX verwenden

- Wenn Sie **Formeln, Diagramme oder mehrere Registerblätter** brauchen.
- Für Berichte, die **von Menschen gelesen** werden, mit visueller Formatierung.
- Wenn Datentypen (Datum, Währung) erhalten bleiben müssen.

Die Einschränkung: größere Dateien, und nicht jedes System importiert XLSX problemlos.

## Kurze Übersicht

| Kriterium | CSV | XLSX |
| --- | --- | --- |
| Formeln und Formatierung | Nein | Ja |
| Mehrere Registerblätter | Nein | Ja |
| Dateigröße | Kleiner | Größer |
| Systemübergreifende Kompatibilität | Sehr hoch | Mittel |
| Ideal für | Datenaustausch | Berichte |

## Häufige Fragen

**Kann ich CSV in XLSX umwandeln und umgekehrt?**
Ja. Excel und die meisten Tabellenkalkulationen öffnen das eine und speichern als das andere. Denken Sie daran, dass das Speichern als CSV Formeln und Formatierung verliert.

**Welches Format benötigt weniger Platz?**
CSV, weil es reiner Text ist. Für große Mengen ohne Formatierung ist es leichter.

**Welches soll ich zum Zusammenführen von Tabellen wählen?**
Verwenden Sie das Format, das das Ziel erwartet. Sehen Sie [wie man Excel-Tabellen zusammenführt](/blog/como-juntar-planilhas-no-excel) und [wie man CSV-Dateien kombiniert](/blog/como-combinar-arquivos-csv).
