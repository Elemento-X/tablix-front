Fusionner deux feuilles de calcul dans Excel semble simple, jusqu'à ce que vous ayez des dizaines de colonnes, des formats différents et des milliers de lignes à concilier. Ce guide présente la méthode manuelle dans Excel et une alternative plus rapide quand le copier-coller devient pénible.

## Avant de commencer : alignez les colonnes

Le secret d'une fusion sans souci est de s'assurer que les deux feuilles parlent la même langue. Vérifiez trois points d'abord :

- **Des en-têtes identiques :** « Courriel » et « E-mail » sont deux colonnes différentes pour Excel. Standardisez les noms.
- **Le même ordre de colonnes :** cela facilite l'empilement des données sans mélanger les champs.
- **Le même format de données :** dates, nombres et texte doivent suivre le même modèle dans les deux fichiers.

Cet alignement évite 90 % des erreurs qui apparaissent après la fusion.

## Étape par étape dans Excel

Pour fusionner deux feuilles en empilant les données (l'une sous l'autre) :

1. Ouvrez la première feuille — elle devient la base du fichier final.
2. Dans la deuxième feuille, sélectionnez les données **sans l'en-tête** (vous en avez déjà un dans la base).
3. Copiez (`Ctrl + C`) et collez dans la première ligne vide sous les données de la base (`Ctrl + V`).
4. Vérifiez que les colonnes sont alignées et supprimez les doublons dans **Données → Supprimer les doublons**.

Pour quelques fichiers et quelques lignes, cela suffit. Le problème surgit quand il faut répéter l'opération chaque semaine, avec des fichiers qui changent de format.

## Quand le copier-coller ne tient plus

Si l'un de ces scénarios vous parle, une approche automatisée vaut le coup :

- Plusieurs fichiers à fusionner d'un coup, pas seulement deux.
- Des colonnes dans un ordre différent dans chaque fichier.
- Le besoin de ne garder que certaines colonnes dans le résultat final.
- Une tâche à répéter fréquemment.

## Questions fréquentes

**Puis-je fusionner plus de deux feuilles à la fois ?**
Oui. Dans Excel, vous empilez une à une ; dans des outils en ligne comme Tablix, vous importez plusieurs fichiers d'un coup et les fusionnez tous en un seul.

**Les colonnes doivent-elles être dans le même ordre ?**
Avec le copier-coller manuel, oui — sinon les données se mélangent. Dans les outils qui reconnaissent les colonnes par nom, l'ordre n'est plus un problème.
